# User Management System - React

User management project built with React + Vite consuming a REST API.

## Tech Stack

- React
- Vite
- Axios
- React Router DOM
- Styled Components

## Features

- Fetch users from API
- Display users list
- Router navigation
- Component-based architecture
- API service layer

## Installation


npm install


## Run


npm run dev


## API


http://localhost:3000/usuarios
